Autocomplete widget

Goal

As a user, suggestions are hinted to me while searching for a country.

As a developer, build the UI (user interface) with keyboard events listening for each key press in the text input field. Send the input value to the autocomplete service and display the response to the user.

Sub-goal

  1. As a developer, listen to the text input and display its value in the suggestion container for every character typed.
  2. As a developer, listen to the text input, pass the value to the autocomplete service, and output the response in the suggestion container.
  3. As a developer, I want the autocomplete suggestions to displayed as a bulleted list
  4. As a developer, I want the autocomplete suggestions to be styled one per line with a surrounding outline with drop shadow.
  5. As a user, I want to be notified when the autocomplete service is slow and forces me to wait
  6. As a user, I want to use my keyboard to highlight autocomplete suggestions.
