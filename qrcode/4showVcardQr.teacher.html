vCard format

BEGIN:VCARD
VERSION:3.0
N:Last;First
FN:First Last
TITLE:Job Title
TEL;CELL:6046822787
EMAIL;WORK;INTERNET:email@vanarts.com
URL:vanarts.com
END:VCARD

Goal

As a user, I want a QR code storing a vCard.

As a dev, get these fields from a HTML form: first & last name, job title, and web address. On submit display the captured data as a QR Code that visualizes the vCard format.
