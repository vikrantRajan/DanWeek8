vCard format

BEGIN:VCARD
VERSION:3.0
N:Last;First
FN:First Last
TITLE:Job Title
TEL;CELL:6046822787
EMAIL;WORK;INTERNET:email@vanarts.com
URL:vanarts.com
END:VCARD

Goal

As a user, I want a vCard generator.

As a dev, get these fields from a HTML form: first & last name, job title, and web address. On submit display the captured data in a vCard format.

Dev steps

  1. Bind submit (event) to form (element)
  2. cancel form action event.preventDefault() or return false;
  3. then update textarea with "hello"
  4. change the textarea to display first name value
  5. change the textarea to display all field's value
  6. change the textarea to display all field's value in vcard format
